  OPTION 1: RENDER.COM (RECOMMENDED - EASIEST)
═══════════════════════════════════════════════════════════════

⏱️  Time: 10 minutes
💰 Cost: $7-14/month (Free tier available)
🔧 Requirements: GitHub account, Browser

STEPS:
------
1. Generate secrets (run on your Mac):
   python3 -c "import secrets; print(secrets.token_urlsafe(32))"
   python3 -c "import secrets; print(secrets.token_urlsafe(32))"
   
2. Push to GitHub:
   cd /Users/alexlim/Desktop/mandate_vault
   git init
   git add .
   git commit -m "Deploy"
   # Create repo on GitHub.com
   git remote add origin https://github.com/YOU/mandate-vault.git
   git push -u origin main

3. Deploy on Render.com:
   • Go to https://render.com
   • Sign up with GitHub
   • New + → PostgreSQL (name: mandate-vault-db)
   • Copy Internal Database URL
   • New + → Web Service
   • Select your repo
   • Environment: Docker
   • Add environment variables:
     - SECRET_KEY=<from step 1>
     - ACP_WEBHOOK_SECRET=<from step 1>
     - DATABASE_URL=<from PostgreSQL>
     - ENVIRONMENT=production
     - DEBUG=false
   • Create Web Service
   
4. Setup database (in Render Shell):
   alembic upgrade head
   python scripts/seed_initial_data.py

DONE! ✅ API live at: https://mandate-vault.onrender.com
