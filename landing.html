Multi-Protocol Authorization

Support both JWT-VC mandates and delegated token authorizations in a single unified API

AP2

Account Provider Protocol 2

JWT-VC based mandates with cryptographic signatures for long-term authorizations

  • RSA/EC signature verification
  • DID-based issuers
  • Self-contained credentials
  • Open Banking compliant
  • Recurring payment mandates
ACP

Authorization Credential Protocol

Delegated token-based protocol with webhook lifecycle management for payments

  • PSP-issued delegations
  • Real-time webhook events
  • HMAC-signed delivery
  • Transaction-specific auth
  • Time-limited permissions
